# 🚀 Star Striker

A modern HTML5 space shooter built with JavaScript and Canvas.

## Features

- 🚀 Multiple game modes
- 👾 Boss battles
- ⚡ Power-ups
- 🏆 Achievements
- 🎮 Endless mode
- 🌌 Dynamic backgrounds
- 💥 Particle effects

## Controls

- Arrow Keys — Move
- Esc — Pause
- R — Restart
